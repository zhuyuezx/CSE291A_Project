# Quoridor Strategy Guide
# This file is read by the AI agent to inform its decisions.
# New insights are appended after each training cycle.

## Core Principles
1. Prioritize path length - shorter path = winning position
2. Walls are precious - each placement should force opponent 2+ extra steps
3. Center control provides more movement options
4. When ahead, push forward. When behind, wall strategically.

## Starting Insights
- Default Strategy: Prioritize path length over wall placement in early game.
